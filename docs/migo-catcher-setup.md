# ミ＝ゴキャッチャー — セットアップ（KP向け）

菌糸コイン・ギフトコード・PCコスメを有効にする手順です。チャウグナー・ランと同じ GAS プロジェクトを使います。

**運用・配布の説明（KP向け）** → `docs/migo-catcher-kp-guide.txt`  
**PL向けの遊び方** → `docs/migo-catcher-pl-guide.txt`

---

## やること一覧

| # | 何をする | 誰が |
|---|---------|------|
| 1 | GAS を最新にして **再デプロイ** | KP（1回） |
| 2 | PL がゲームを遊ぶ / KP がコード発行 | 誰でも |
| 3 | シート `MIGO_*` が自動作成される | **自動** |

ゲームの URL 設定は不要です。`js/config.js` の GAS をそのまま使います。

---

## 手順 1 — GAS を更新して再デプロイ

1. [Google Apps Script](https://script.google.com/) で **NPC フォーム用プロジェクト** を開く
2. `docs/gas-npc-form.gs` の中身を **すべて** `Code.gs` にコピーして保存
3. **デプロイ → デプロイを管理 → 編集（鉛筆）**
4. バージョン: **新バージョン** → **デプロイ**

### 確認 URL（`js/config.js` の baseUrl と同じベース）

残高（0でもOK）:

```
.../exec?type=migo-balance&player_name=テスト
```

→ `{"ok":true,"balance":0,...}`

ランキング:

```
.../exec?type=migo-ranking
```

→ `[]` または配列

`unknown type` が出たら再デプロイがまだです。

---

## 手順 2 — 自動でできるシート

初回利用時に同じスプレッドシートへ自動作成されます。

| シート | 用途 |
|--------|------|
| `MIGO_COINS` | プレイヤー名 → コイン残高 |
| `MIGO_GIFT_CODES` | KP発行のギフトコード |
| `MIGO_UNLOCKS` | PCごとのコスメ解放 |
| `MIGO_PLAYS` | プレイ履歴・GETランキング |

手動で作る必要はありません。

## コインの入手方法

| 経路 | 内容 |
|------|------|
| **KPギフトコード** | KPページ「ミ＝ゴギフトコード」から発行 → PLがゲームで換金 |
| **KP直接付与** | KPページ「オマケ通貨」→ PL別残高から付与 |
| **チャウグナー・ラン** | 1000m ごとに 1 コイン（1プレイ最大 2 コイン） |

## KPでの管理（オマケ通貨）

KPページの **「オマケ通貨（菌糸コイン）」** セクションで:

| 機能 | 用途 |
|------|------|
| **PL別コイン残高** | 誰がいくら持っているか一覧 |
| **コイン直接付与** | シナリオ後のお疲れ様、調整 |
| **名前変更（統合）** | プレイヤー名変更時に残高をまとめる |
| **ギフトコード発行** | PLに渡す換金コード |

## PL名（プレイヤー名）について

- **正** は PC登録シート（`PCS`）の `player_name` です
- コイン残高もこの名前に紐づきます（PL単位）
- ミ＝ゴ・チャウグナーとも、ゲームで入力する名前は **これと完全一致** が必要
- プレイヤー名を変えるときは **PCフォームの編集** と **KPでの残高統合** の両方を行ってください

ミ＝ゴキャッチャーは **コイン 1 枚 = 1 回プレイ** です。

---

## PL の遊び方

1. ホーム「オマケ」→ **ミ＝ゴキャッチャー**
2. **プレイヤー名**（PC登録と同じ）を入力
3. **飾る PC** を選択
4. ギフトコードがあれば換金
5. **コイン1枚で筐体を起動** → 1回だけ爪操作
6. GET した景品は **その PC の一覧・詳細** にコスメとして表示される

## うまくいかないとき

| 症状 | 対処 |
|------|------|
| コインが `—` のまま | GAS 未デプロイ or ページ再読込 |
| 登録PCが見つからない | プレイヤー名が PCS の `player_name` と完全一致しているか確認 |
| 換金できない | コードの期限・使用回数を確認（`MIGO_GIFT_CODES` シート） |
| PCにコスメが出ない | サイトを再読込（`pcs` API に `cosmetics` が付く） |
