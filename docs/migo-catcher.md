# ミ＝ゴキャッチャー — コンセプト・ルール

## 名称について

| 表記 | 意味 |
|------|------|
| **ミ＝ゴ** | クトゥルフ神話の地球外生命体 **Mi-go**（ユゴスよりの菌類） |
| **ミ＝ゴキャッチャー** | ミ＝ゴの鉤爪で景品を掴む筐体ゲーム |

**「ミニゴー」「ミニゴ」とは無関係。** `＝` は Mi-go の日本語表記の一部。

技術パス `games/mini-claw.*` はリポジトリ内のファイル名のみ（URLは変更不要）。

---

## 世界観（ゲーム内設定）

ヨコフォリアの地下に、**薄桃色の甲殻を持つミ＝ゴ**が運営する「菌糸チケット式」の景品筐体がある。

- 筐体のクレーンではなく、**ミ＝ゴの鉤爪の足**が左右に動き、降下して掴む
- 光は弱め（ミ＝ゴは光を苦手とする）
- 掴んだ景品は **選んだPCの表示用コスメ** として残る（実装予定）
- 写真に写りにくい存在だが、そのPCの台帳ページだけは地球側に「残る」——そういうオマケ設定

---

## システム全体（オマケ通貨 — 菌糸コイン）

```
【コイン入手】
  ├ KP配布（ギフトコード or 直接付与）
  └ チャウグナー・ラン（スコア報酬）

【コインの単位】
  プレイヤー名（PC登録の player_name）ごとに1つの残高
  → チャウグナーでもミ＝ゴでも同じコインを使う

【ミ＝ゴでの消費】
  1. 飾る PC を選択
  2. ギフトコード入力（持っていれば換金）
  3. コイン1枚消費 → ミ＝ゴキャッチャー1回
  4. 景品GET → **選択したPC** にコスメ付与

【サイト上の見え方】
  そのPCの一覧カード・詳細ページだけが豪華になる（他のPCやNPCには影響しない）
  誰が閲覧しても同じ装飾が見える
```

ログインなし。プレイヤー名は PC 登録（`player_name`）と照合。チャウグナー・ランと同じ信頼モデル。

## プレイルール（筐体）

| 操作 | 内容 |
|------|------|
| **◀ ▶** | ミ＝ゴの腕（爪）を左右に移動 |
| **▼** | 腕を降下し、鉤爪で掴む |
| 位置 | 景品の真上に近いほど成功率アップ |
| 1回 | コイン1枚（実装予定） |

### 難易度（景品ごと）

| 重さ | 成功率目安 | 例 |
|------|-----------|-----|
| 軽い | 高め | 菌糸、隕石片 |
| 重い | 低め | エーテル結晶、第三の眼 |

### 失敗時

- 爪が空振り → コインは消費済み、景品なし
- 掴んだが上昇中に落下（将来実装可）

## 景品 ↔ PCコスメ対応表（初版）

| 景品 | cosmetic_id | PCでの効果 |
|------|-------------|-----------|
| ユゴス菌糸 🍄 | `frame-fungal` | 枠：菌糸模様 |
| 隕石片 ☄️ | `bg-nebula` | 背景：星雲 |
| 禁断フィルム 📼 | `title-whisper` | 称号：囁きの観測者 |
| 地球製骨片 🦴 | `frame-bone` | 枠：骨格風 |
| 第三の眼 👁 | `fx-glimpse` | 微光エフェクト |
| 暗黒の欠片 🌑 | `bg-void` | 背景：深闇 |
| エーテル結晶 🛸 | `frame-ether` | 枠：エーテル色 |
| 鉤爪の欠片 🔗 | `title-migo` | 称号：菌類の取引人 |

装備スロット: **枠 / 背景 / 称号 / 光** の4つ。

- **コレクション** — 取った景品はすべて `MIGO_UNLOCKS` に残る（コンプ判定用）
- **表示** — 同スロットは**最新の1つだけ**台帳に反映（古い枠はコレクションには残る）
- **コンプ** — 8種すべてを一度でも取ったら `frame-migo-wing` を自動付与

## データ（スプシ・GAS予定）

| シート | 用途 |
|--------|------|
| `MIGO_COINS` | player_name → 残高 |
| `MIGO_GIFT_CODES` | ギフトコード |
| `MIGO_UNLOCKS` | pc_id + cosmetic_id |
| `MIGO_PLAYS` | ランキング用 GET数 |

API例: `migo-balance`, `migo-redeem`, `migo-play`, `migo-unlock`, `migo-ranking`

## 参考（ミ＝ゴ / Mi-go）

- ユゴスよりの菌類。薄桃色の甲殻、蝙蝠状の翼、多数の鉤爪の足
- 光を苦手とする。テレパシー・ブザー音で意思疎通
- 『闇に囁くもの』などに登場

ゲームはホラー演出は控えめ・オマケの範囲に留める。
