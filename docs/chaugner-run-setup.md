# チャウグナー・ラン — ランキング設定（やること3つ）

スコアをスプレッドシートに保存して TOP3 を表示する手順です。

---

## やること一覧

| # | 何をする | 誰が |
|---|---------|------|
| 1 | GAS のコードを最新にして **再デプロイ** | KP（1回だけ） |
| 2 | ゲームを遊んでスコアを出す | 誰でも |
| 3 | スプレッドシートに `CHAUGNER_SCORES` シートができる | **自動** |

ゲーム側の URL 設定は **不要** です。NPC と同じ `js/config.js` の GAS を自動で使います。

---

## 手順 1 — GAS を更新して再デプロイ（KP向け・1回だけ）

1. [Google Apps Script](https://script.google.com/) で **NPC フォーム用プロジェクト** を開く
2. `docs/gas-npc-form.gs` の中身を **すべて** `Code.gs` にコピーして保存
3. **デプロイ → デプロイを管理 → 編集（鉛筆マーク）**
4. バージョン: **新バージョン** → **デプロイ**

### できているか確認

ブラウザで次の URL を開く（`...` の部分は `js/config.js` の `baseUrl` と同じ）:

```
https://script.google.com/macros/s/AKfycbw20JiLGzzdC2m7uOgP14RecbtwFhbHUBuO-emsrDThF-YRsB8Ux60SQPOlRU5yXquw/exec?type=chaugner-ranking
```

- `[]` または `[{...}]` が表示されれば OK
- `unknown type` と出たら → 手順1のコード貼り付け or 再デプロイがまだ

テスト保存:

```
.../exec?type=chaugner-score&name=テスト&score=100
```

→ `{"ok":true,...}` なら OK

---

## 手順 2 — シートについて

**自分でシートを作る必要はありません。**

初めてスコアが保存されたとき、同じスプレッドシートに `CHAUGNER_SCORES` というシートが **自動で** できます。

中身のイメージ:

| name | score | created_at |
|------|-------|------------|
| トトロン | 193 | 2026/06/16 15:30:00 |

---

## ランキングのルール

- 歴代 **TOP3** を表示
- スコアが高い順（同点なら古い記録が上位）

---

## うまくいかないとき

| 画面に出る表示 | 原因 | 対処 |
|--------------|------|------|
| ランキングは準備中です | GAS URL が読めていない | 通常は起きない。ページを再読み込み |
| まだ記録がありません | 正常。誰も遊んでいない | ゲームを1回クリアして終了 |
| ランキング取得に失敗 | GAS が古い or 未デプロイ | 手順1をやり直す |
| スコアの保存に失敗 | 同上 | 手順1 + 上のテスト URL で確認 |
