# PC 登録フォーム — セットアップ手順（PL向け）

## プロジェクトの名前

| 名前 | 実体 | やること |
|------|------|----------|
| **DBPJ** | スプレッドシート | データが溜まる |
| **NPCPJ** | `docs/gas-npc-form.gs` | サイト API・デプロイ |
| **PC PJ** | `docs/gas-pc-form-bound.gs` | **PL の PC 登録 → PCS 転記** |

> **登録は PL が行います。** KP は一覧・表示設定（非表示）のみ。  
> 関連シナリオ・連絡可能NPCは **シナリオ / NPC フォームから PC名で自動紐づけ** されます。

---

## フォーム質問（この順・この文言）

### フォーム全体

**タイトル:** `YOKOFOLIA PC登録`

**説明文:**
```
YOKOFOLIA ふわっと住民台帳に、あなたの PC を登録するフォームです。

【入力するのは3つだけ】
・PC名（キャラクター名）
・プレイヤー名（あなたの名前・ハンドルネーム）
・キャラシURL（Googleスプレッドシート、PDF、note など）

関連シナリオや NPC とのつながりは、KP がシナリオ・NPC フォームに
PC名を書くと自動で表示されます。ここでは書かなくて大丈夫です。

送信後も編集リンクから内容を直せます。
```

### 質問

| # | タイプ | 質問文 | 必須 | 説明 |
|---|--------|--------|------|------|
| 1 | 短い回答 | PC名 | ✅ | 台帳に表示されるキャラクター名です。 |
| 2 | 短い回答 | プレイヤー名 | ✅ | あなたの名前や Discord 名など。 |
| 3 | 短い回答 | キャラシURL | ✅ | `https://` で始まる URL を入力してください。 |

---

## GAS セットアップ

1. PC 入力フォーム → スクリプトエディタ（PC PJ）
2. `docs/gas-pc-form-bound.gs` を貼り付け
3. トリガー: `onPcFormSubmit` / フォームから / 送信時
4. 回答先は **NPC と同じ DBPJ**

## NPCPJ 再デプロイ

`?type=pcs` / `pc-visibility` を有効化するため、NPCPJ を再デプロイしてください。

## リンク設定

フォーム URL を `js/links.js` の `pcForm` に設定すると、PL の PC ページに「PCを登録する」ボタンが表示されます。
